Name Mode

This toggle button in the Read window lower toolbar sets the Name Mode:

Name Mode

Purpose

When Name Mode is active, transliterations for the names/epithets of God in the Tanakh (Hebrew Text) are substituted for the original KJV text.

Scope

The substitutions provided in Name Mode are as follows:

Limitations

Name Mode only affects verse text in the Read pane.

The Search Lookup pane only searches the original KJV text. For example, you can not search for Yahweh, El, etc.

The Search Result pane uses the original KJV text.

The Strong Verse and Result panes, as well as word summaries in the Definition pane, use the KJV original text.
